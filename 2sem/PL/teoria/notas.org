** Expressões regulares
+ a :: pesquisar a
+ a.b :: a seguido de b
+ a|b :: a ou b
+ () :: agrupar

  Exemplo:
  (0|..|9) -> número entre 0 e 9
+ a? :: opcional ter a
+ * :: 0 ou mais vezes o q vier antes
+ + :: 1 ou mais vezes o q vier antes
+ [] ::
